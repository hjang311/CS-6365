## 📖 Course Overview

> [!NOTE]
> CS 4365/6365 focuses on the fundamental conceptual pillars of enterprise IT, complemented by scalable implementation of Quality of Service (QoS) dimensions in the cloud. It bridges the gap between horizontal cloud scalability, transactions, services, and AI/ML.

### **Two Core Learning Objectives**
1. **Technical Information:** transaction processing, workflow & business process management, services computing, cloud computing, and AI/ML.
2. **Autonomous Learning Skills:** developing Level 1 (Fact vs. Opinion/Logical Reasoning), Level 2 (Science Skills: Theory in practice, Check assumptions, Reapply good ideas), and Level 3 (Trend Recognition: Tech push / App pull, System evolution cycles).

## 🚀 Project Focus: NORP (Non-Profit Organizations)

Our semester project will focus on **NORP**, a structured project that applies Enterprise Computing principles to the ecosystem of non-profit organizations. 

We will apply the following core pillars of enterprise computing to build a robust, scalable, and secure system:
* **Transactions (MT):** Ensuring the "Conservation of Money" for donor transactions using ACID transaction guarantees.
* **QoS & Cloud (MQ):** Managing statistical QoS (P99 performance, 7x24 continuity, high availability) for fundraising drives.
* **Services & Workflows (MS):** Orchestrating donor-to-needs matching workflows and building secure microservices.
* **AI/ML (MA):** Incorporating situational awareness and modern AI/LLM tools to optimize resource allocation and donor engagement.
