# Screenshots Directory

This directory contains evidence screenshots captured during the reproducibility exercise.

## Naming Convention

Please name screenshots using the following convention:

```
<step_number>_<description>.png
```

**Examples:**
- `01_repo_clone.png`
- `02_env_setup.png`
- `03_ca1_output.png`
- `04_ca2_extraction_results.png`
- `05_ca3_analysis_chart.png`
- `06_rag_query_result.png`
